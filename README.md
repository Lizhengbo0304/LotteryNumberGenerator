# 彩票选号助手 | Lottery Number Generator

## 项目简介 | Introduction
一个现代化的彩票号码生成工具，支持多种彩票类型和投注方式，提供直观的用户界面和便捷的操作体验。

A modern lottery number generator that supports various lottery types and betting methods, offering an intuitive user interface and convenient operation experience.

## 主要功能 | Features
- ? **多彩种支持 | Multiple Lottery Types**
  - 大乐透 (DaLeTou)
  - 双色球 (ShuangSeQiu)

- ? **多种投注模式 | Multiple Betting Modes**
  - 普通投注 (Simple)
  - 复式投注 (Multiple)
  - 胆拖投注 (Dantuo)

- ? **智能选号 | Smart Number Selection**
  - 随机选号
  - 智能过滤
  - 历史号码分析

- ? **深色模式 | Dark Mode**
  - 自动适应系统主题
  - 手动切换主题

- ? **导出功能 | Export Feature**
  - 支持导出为TXT格式
  - 便于保存和打印

## 技术栈 | Tech Stack

- Tailwind CSS 3.0
- Vanilla JavaScript
- Font Awesome 6.4.0

## 使用说明 | Usage
1. 选择彩票类型（大乐透/双色球）
2. 选择投注方式（普通/复式/胆拖）
3. 设置选号参数
4. 点击生成按钮获取号码
5. 可选择导出或重新生成
