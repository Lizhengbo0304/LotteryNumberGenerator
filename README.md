# 彩票选号助手 Lottery Number Generator

## 功能特点 Features
- **支持彩票类型**: 大乐透/双色球 (Daletou/Shuangseqiu)
- **投注模式**: 普通/复式/胆拖 (Simple/Multiple/Dantuo)
- **智能生成**: 根据预设规则自动生成有效号码
- **主题切换**: 支持深色/浅色模式 (Dark/Light Theme)
- **批量导出**: 可导出.txt格式的号码清单

## 访问地址 URL
https://lizhengbo0304.github.io/LotteryNumberGenerator/

## 技术栈 Tech Stack
- HTML5/CSS3
- Tailwind CSS 3.0
- Vanilla JavaScript

## 使用说明 Usage
1. 选择彩票类型
2. 设置投注参数
3. 点击"立即生成"按钮
4. 导出或复制号码
